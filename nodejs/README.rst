===================
Node.js Backend App
===================

A Node.js backend app using express and node-crate.

Prerequisites
=============

* Nodejs_
* npm_

Setup
=====

Install dependencies:

.. code-block:: sh

    $ cd nodejs
    $ npm install

Run
===

You can run the app like so:

.. code-block:: sh

    $ npm start


.. _Nodejs: https://nodejs.org/en/
.. _npm: https://www.npmjs.com/get-npm
